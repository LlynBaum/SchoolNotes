# Multipass

Command and other things about Multipass

## Commands

| Befehlt | Beschreibung |   
| :------ | :---------- |
| `multipass list` | lists all instances |  
| `multipass launch` | Creates and Starts new Ubuntu instance |  
| `multipass shell <name>` | open shell for the VM |
